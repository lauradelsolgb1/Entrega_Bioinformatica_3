# Entrega_Bioinformatica_3
Entrega 3: Ortología y Filogenia.

1. Ortólogos
2. Alineamiento múltiple con MAFFT. Seleccionar e incluir un Outgrup en el alineamiento.
3. Filogenia con IQ-TREE para la construcción del árbol e identificar el mejor modelo de sustitución.

Entregables:
- Archivos FASTA de ortólogos. Puede ser un multifasta
- Alineamiento múltiple (FASTA).
- Árbol filogenético en formato Newick y visualizado bonito
- Informe PDF con 
  - Imagen del árbol.
  - Interpretación biológica de relaciones filogenéticas.
- Preguntas:
  - ¿Que diferencia hay entre ortólogos y parálogos?
  - ¿Por qué se debe incluir un outgoup en el árbol? ¿En qué casos se debe hacer?
  - ¿Cuál es la diferencia entre el enfoque frecuentista y bayesiano para la reconstrucción de árboles filogenéticos?
